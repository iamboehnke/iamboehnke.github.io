Analyzing the Impact of News on Stock Markets

This project uses TensorFlow and machine learning techniques to predict stock prices based on news sentiment. Below is a summary of the steps involved:

Step 1: Data Collection

I gathered news articles from the News API and stock data from Alpha Vantage. Here’s a sample of the data collection code:

import requests
news_api_url = "https://newsapi.org/v2/everything"
params = {
    "q": "stock market",
    "apiKey": "YOUR_API_KEY"
}
response = requests.get(news_api_url, params=params)
articles = response.json()["articles"]

Step 2: Data Preprocessing

The news articles were tokenized and vectorized, while stock prices were normalized to match the news data:

from sklearn.feature_extraction.text import TfidfVectorizer
vectorizer = TfidfVectorizer(stop_words='english')
news_data = vectorizer.fit_transform([article["title"] for article in articles])

Step 3: Model Building

I built a neural network with TensorFlow that combines news sentiment with stock prices:

import tensorflow as tf
model = tf.keras.Sequential([
    tf.keras.layers.Dense(128, activation='relu', input_dim=news_data.shape[1]),
    tf.keras.layers.LSTM(64),
    tf.keras.layers.Dense(1)
])
model.compile(optimizer='adam', loss='mean_squared_error')

Step 4: Model Training

I trained the model using historical data and evaluated it using the Mean Absolute Error metric:

history = model.fit(news_data, stock_prices, epochs=10, batch_size=32)
